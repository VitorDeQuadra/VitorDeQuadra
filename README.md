### 👋 Sobre mim

Sou desenvolvedor fullstack e fundador da **[DeQuadra](https://dequadraapps.com.br)** — uma software house focada em criar sites, sistemas web e aplicativos mobile para empresas de todo o Brasil, com base em Pato Branco/PR.

Atualmente cursando **Análise e Desenvolvimento de Sistemas** e **Engenharia de Software**, e me aperfeiçoando em **UX/UI Design** para entregar produtos que combinam código limpo com experiências memoráveis.
